Software Prototype

Our proof-of-concept prototype consists of a Click-based router, network support services such as the name resolution service, and a host protocol stack and service API for accessing the network. The clean slate architecture implements a new narrow waist based on GUID (similar to IP for current Internet), which forms the definitive network identifier for all network-attached objects in MobilityFirst. A network of MobilityFirst routers, hosts and services require layer-2 access on wired and wireless networks and supports deployments on Ethernet, WiFi and WiMAX access technologies. Some components, however, such as the global name resolution service (GNRS) are easily adapted to run standalone on IP networks.

Release

A first public release of the architecture prototype has been made available in December 2014. For more details head to the Project Wiki page.
